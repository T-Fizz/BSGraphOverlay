This is an overlay for Beat Saber. Once the page is loaded, you will be prompted to enter in your name to display (or leave empty to display nothing).

To use this overlay in OBS, copy this link and add as a browser source: https://t-fizz.github.io/BSGraphOverlay/

Color for the graph can be chosen by entering in any CSS color, rgb values, or hex values. In order for the color to be anything other than default, they must be entered exactly like the following color examples:

rgb(0, 23, 255)
#336699
ffee66
fb0
red
darkblue
cadet blue
